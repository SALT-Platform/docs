## Project Silverado

Project Silverado is SALT Lending's de-SPAC merger transaction with LightWave Acquisition Corp (NASDAQ: LWAV). The transaction positions SALT as a publicly-listed digital asset lending platform.

## Content Governance

The presentation implements a three-tier content governance model:

- **Deck Content** — Visible to all authenticated users when published
- **Verbal Only** — Talking points visible only to admin users (blue badge)
- **Internal Annotations** — BTIG review notes visible only to admin users (gray badge)
