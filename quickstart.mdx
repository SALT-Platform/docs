## Prerequisites

- Git access to [SALT-Dev-SO](https://github.com/SALT-Dev-SO) organization
- Python 3.9+ for D&O Dashboard development
- Node.js 18+ for Mintlify CLI (optional, for local docs preview)

## Clone the Repositories

```bash
# D&O Dashboard
git clone https://github.com/SALT-Dev-SO/SALT-Directors-and-Officers-Dashboard.git
cd SALT-Directors-and-Officers-Dashboard
pip install -r requirements.txt

# Documentation
git clone https://github.com/equanaut-sha-w1/docs.git
```

## Run the D&O Dashboard Locally

```bash
cd SALT-Directors-and-Officers-Dashboard
export SECRET_KEY="your-dev-secret"
export FLASK_ENV=development
python3 app.py
# Dashboard available at http://localhost:5001
```

## Preview Documentation Locally

```bash
cd docs
npx mintlify dev
# Docs preview at http://localhost:3000
```
