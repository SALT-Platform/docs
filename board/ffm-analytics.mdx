## Financial Forecast Model (Spec 016)

The FFM Analytics module parses the Spectrum Financial Forecast Model (an Excel workbook) into structured JSON snapshots, then renders four interactive dashboard pages.

## Architecture

```
Spectrum XLSX → openpyxl Parser → Pydantic Models → JSON Snapshots → Flask API → Dashboard Pages
```

## Data Model

| Entity | Description |
|--------|-------------|
| `FFMSnapshot` | Root model containing all parsed sections |
| `SummaryIS` | Income statement — revenue lines, EBITDA, margins |
| `KeyAssumptions` | 5 lending segments with loans receivable and revenue arrays |
| `Valuation` | Close-date scenarios with NTM multiples and implied EVs |
| `BTCImpact` | Toggle (ON/OFF), CAGR, and BTC-adjusted scenarios |
| `SPACSspend` | Treasury allocation and de-SPAC cash deployment |
| `Headcount` | Department-level headcount projections (5 years) |

## Snapshot Management

FFM snapshots are stored as JSON files in `data/ffm_snapshots/` with timestamps. The latest snapshot is automatically selected for rendering. Multiple snapshots enable historical comparison.
